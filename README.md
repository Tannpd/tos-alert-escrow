# TOS-Alert Escrow — Decentralized Anti-TOS Abuse Protocol

TOS-Alert Escrow is a decentralized subscription escrow and legal defense fund. It holds user budget pools (subscription fees or community treasuries) and continuously monitors the Terms of Service (TOS) of a platform using GenLayer AI Validators. If a breach of the community's predefined Red Line Rules is confirmed, payments to the platform are permanently frozen, and users can rage-quit and claim proportional refunds.

### Why TOS-Alert DIES without GenLayer
> **Traditional blockchains (like Ethereum or Solana) cannot natively read live legal document pages or run natural language evaluations, meaning TOS-Alert would die without GenLayer's unique ability to scrape platform legal URLs and run decentralized, semantic consensus checks to verify qualitative rules on-chain.**

---

## Features
- **TOS Domain Verification**: Validates that submitted TOS URLs match the whitelisted domain to prevent user spoofing.
- **AI Legal Counsel**: Scrapes legal text using `web.render` and prompts `exec_prompt` to analyze the clauses against predefined Red Line Rules.
- **Semantic Legal Consensus**: A custom validator ensures consensus is reached on the logical verdict (`breach_found` boolean status) rather than comparing raw strings or cited paragraphs.
- **Proportional Rage-Quit Refunds**: Calculates claims proportionally based on remaining balance and historical stakes, ensuring fairness if partial payments were already released to the platform.
- **Secure Platform Payment Releases**: Allows the community owner to release payments periodically, which is permanently frozen if a breach is detected.

## 1. How to Deploy the Intelligent Contract

To deploy your contract using **GenLayer Studio**:

1. Open **[GenLayer Studio](https://studio.genlayer.com/)**.
2. Create or upload a new file named `tos_alert.py` containing the code in `contracts/tos_alert.py`.
3. Check the compiler detection header at the top:
   ```python
   # v0.2.16
   # { "Depends": "py-genlayer:1jb45aa8ynh2a9c9xn3b7qqh8sm5q93hwfp7jqmwsfhh8jpz09h6" }
   ```
4. Compile the file and select `Contract` in the compilation output panel.
5. Click **Deploy**.
6. Once deployed successfully, copy the **Contract Address** generated by GenLayer Studio.

## 2. How to Run the Frontend

### Prerequisites
Make sure you have **Node.js** (v18 or higher) and **npm** installed on your machine.

### Installation Steps

1. Navigate to the `frontend` folder:
   ```bash
   cd D:\Gen\TOSAlertEscrow\frontend
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Configure your contract address. Open the `D:\Gen\TOSAlertEscrow\frontend\.env` file and paste your deployed contract address:
   ```env
   VITE_CONTRACT_ADDRESS=0xYourDeployedContractAddressHere
   ```

4. Start the local Vite development server:
   ```bash
   npm run dev
   ```

5. Open the local port in your browser (usually `http://localhost:3000`).

## 3. How to Use the dApp

1. **Connect Account**: 
   Click **Connect Wallet**. The app will connect to your browser extension or generate a secure fallback local account.
2. **Initialize Escrow**:
   *   Configure the platform's whitelisted domain (e.g. `slack.com`).
   *   Provide the platform's recipient wallet address.
   *   Define your community's **Red Line Rules** (e.g. forbidding AI training on user data).
   *   Click **Initialize Contract**.
3. **Escrow Pool Deposits**:
   *   Community members deposit GEN into the escrow pool using the **Deposit Funds** panel.
4. **Platform Payment Release**:
   *   While secure, the manager can release subscription budgets to the platform via the **Release Escrow Payout** form.
5. **Trigger AI Legal Audit**:
   *   Anyone can paste a live TOS URL belonging to the whitelisted domain (e.g. `https://www.slack.com/terms`) and click **Run AI Legal Audit**.
   *   AI Legal Counsel validators scrape and audit the text.
6. **Breach Action / Rage Quit**:
   *   If the AI consensus detects a privacy violation, the contract locks payments to the platform.
   *   The dashboard reveals the AI critique, specific violating clause, and enables the **Rage Quit & Withdraw** button. Depositors claim their proportional refund immediately.
